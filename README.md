# CS471 – Web Technologies  
## University Labs Repository

This repository contains all my lab work for the **CS471 – Web Technologies** course at my university.  

### **Course Overview**
The CS471 course focuses on:
- Understanding fundamental web technologies and protocols.
- Learning how the web works, from low-level protocols like **HTTP**, **TCP/IP**, and **UDP** to higher-level web development concepts.
- Gaining hands-on experience through practical labs using tools like **Wireshark**, **Git**, and other web-related technologies.

### **Repository Structure**
Each lab has its own folder, containing the required files, screenshots, and documentation.
